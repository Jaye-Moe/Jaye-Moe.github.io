I am a CPA that specializes in taxation, mostly working with flow-through entities and high net worth individuals. In my free time I enjoy programming in Python and Java. Below are a few of the programs I have created.

Snacker

A clone of the classic pellet eating game. This is my first game to read a text file that is used to generate a map. I plan on eventually adding additional levels, increases to the speed of the fireballs as points are earned, and having the fireballs change directions instead of just moving in straight lines.

Snake

A clone of the classic snake game. I plan on eventually adding multiplayer mode, additional levels, and increases to the speed of the snake as points are earned.

Wordle Clone

A clone of the word game Wordle. Use the letter keys to enter letters and the enter key to submit a guess. Yellow tiles are correct letters in the incorrect place, green tiles are correct letters in the correct place.

2048 Clone

A clone of the classic numbers game 2048. The game indicates when no more moves are available and allow you to reset the game. I have not yet added a message indicating that the game has been won once a 2048 tile is made. While debugging the program I added a method that allows the user to generate a tile by pressing the space bar. This method has been left in the program and by used as a "cheat code".

Animated Dice Roller

An animated dice roller. Click on each individual die to roll it, or click on the button to roll all the dice. Pseudo-animation was achieved having the dice picture update every 50 milliseconds when rolling.

Tip Calculator

A simple tip calculator. This is my fourth Java project after completing the lessons from the MOOC Java Programming course from the University of Helsinki.

Tic-tac-toe

A tic-tic-toe game. Click on the square where you want to place your letter. Currently two-player only. Planning on eventually updated to include AI for one-player mode.

Pong clone

A clone of the classic game pong. This is my first Java project after completing the lessons from the MOOC Java Programming course from the University of Helsinki.
